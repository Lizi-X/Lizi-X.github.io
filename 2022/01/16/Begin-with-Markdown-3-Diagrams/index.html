Gantt

  • Start with ```mermaid. Four elements:

    • type: gantt

    • title: no ‘:’ after ‘title’

    • dataFormat: no ‘:’ after ‘dataFormat’

    • section: no ‘:’ after ‘section’

      • task name: no space after ‘task:’

      • state: 4states

        state meaning
        done did/have done
        active be doing
        blank be to do
